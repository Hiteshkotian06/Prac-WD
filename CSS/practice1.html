CSS things

There is inline , Internal and External Css

I've shown the internal in the head above using style where i used id,class and all element. Thing to know if i use ID or class then the style apply to only those and no effect of all element tag

For Inline css we add styles in the body only like we did in HTML practice files

For Internal css we add styles in the head tag of the HTML file

For External css we create a separate file and link it to the HTML file

In the style sheet for calling ID we use ( # )

In the style sheet for calling class we use ( . )

For calling all elements we use ( * )
